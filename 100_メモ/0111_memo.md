# 明日の依頼事項①
    WSL環境でPlaywright（Chromium）を動作させるために必要な依存関係をまとめます。   

    インストールが必要なもの

    方法1: Playwrightの自動インストールコマンド（推奨）

    sudo npx playwright install-deps chromium

    これを実行すると、以下のパッケージが自動的にインストールされます。

    方法2: 手動でaptインストールする場合

    以下のパッケージが必要です：

    sudo apt-get update
    sudo apt-get install -y \
        libnss3 \
        libnspr4 \
        libatk1.0-0 \
        libatk-bridge2.0-0 \
        libcups2 \
        libdrm2 \
        libxkbcommon0 \
        libxcomposite1 \
        libxdamage1 \
        libxfixes3 \
        libxrandr2 \
        libgbm1 \
        libasound2 \
        libpango-1.0-0 \
        libcairo2 \
        libatspi2.0-0

    ---
    管理者への依頼文テンプレート

    件名: WSL環境でのPlaywright実行に必要な依存関係のインストール依頼

    以下のコマンドの実行をお願いします：

    sudo npx playwright install-deps chromium

    または、以下のaptパッケージのインストール：

    sudo apt-get update && sudo apt-get install -y libnss3 libnspr4 libatk1.0-0 libatk-bridge2.0-0 libcups2 libdrm2 libxkbcommon0 libxcomposite1 libxdamage1 libxfixes3 libxrandr2 libgbm1 libasound2 libpango-1.0-0 libcairo2 libatspi2.0-0

    用途: Webスクレイピング（Playwrightによるブラウザ自動操作）
