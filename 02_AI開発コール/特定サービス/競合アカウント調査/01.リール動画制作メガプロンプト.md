# Instagramリール動画制作メガプロンプト
## 形式・スタイル・表現方法完全再現ガイド

---

## 📋 概要

このメガプロンプトは、Instagramリール動画における**形式・スタイル・表現方法**のすべての要素を、異なる内容で完全に再現するための詳細な指示書です。このガイドに従うことで、元のリール動画と同じ構成、視覚効果、テンポ感を持ちながら、まったく異なるテーマ・内容のリール動画を制作できます。

---

## 🎬 動画全体の構成フレームワーク

### 基本スペック

| 項目 | 仕様 |
|------|------|
| **アスペクト比** | 9:16（縦型フルスクリーン） |
| **推定総尺** | 18-24秒 |
| **シーン数** | 6シーン |
| **各シーン長** | 3-4秒程度 |
| **フレームレート** | 30fps（標準） |
| **解像度** | Instagram標準品質 |

### 全体的なストーリー構成

このリール動画は、**AIDA（注意・興味・欲望・行動）モデル**に基づいた以下の流れで構成されています。

**Phase 1：Attention（注意喚起）** → **Phase 2：Interest（興味喚起）** → **Phase 3：Desire（欲望醸成）** → **Phase 4：Action（行動喚起）**

## 🎥 シーン別詳細仕様

### シーン1：導入部（感情的訴求）
**目的**：視聴者の注意を引き、感情的に訴求する

#### 背景・ロケーション
- **環境**：屋外（自然環境）
- **背景要素**：ぼかされた緑色の自然背景
- **ボケ効果**：背景を強くぼかして、人物を強調
- **照明**：自然光（昼間の撮影）
- **天候**：晴天
- **時間帯**：昼間（自然光が豊富な時間）

#### 人物表現
- **表情**：笑顔（歯を見せた自然な笑み）
- **視線**：カメラ目線（視聴者に直接訴求）
- **ジェスチャー**：自然な立ち姿勢
- **服装**：青系のシャツ（落ち着いた色合い）
- **髪型**：自然なスタイル
- **雰囲気**：ポジティブ、親近感のある印象

#### テキスト表示
- **位置**：画面下部中央
- **背景**：半透明の黒背景（RGB: 0, 0, 0, 透明度: 60-70%）
- **テキスト内容**：ポジティブな感情表現（例：「本気でテンション上がってる」）
- **フォント**：白色（RGB: 255, 255, 255）、サンセリフ体、太めのウェイト（Bold）
- **サイズ**：画面の約15-20%の高さ
- **改行**：2行表示
- **レイアウト**：中央揃え
- **効果**：テキスト出現時にフェードイン効果

#### 色調・グレーディング
- **彩度**：中程度（自然な見た目を維持）
- **コントラスト**：適度に高い（メリハリのある映像）
- **明度**：明るい（昼間の屋外撮影）
- **フィルター**：軽いフィルター効果（自然さを損なわない程度）
- **色温度**：暖色系（自然な昼間の色温度）

#### 音声・効果音
- **BGM**：テンポ中程度のオリジナル音源
- **効果音**：テキスト出現時に軽い効果音（ポップ音など）
- **音量**：適度（視聴者が快適に視聴可能）

#### シーン長
- **推定尺**：3-4秒

- **影**：最小限（顔全体が明るく見える）

#### 人物表現
- **人物**：眼鏡をかけた男性（知識的、信頼感のあるイメージ）

- **視線**：カメラ目線（視聴者に直接訴求）

## 🎨 テキスト表示の統一仕様

### テキストの基本ルール

#### 配置・レイアウト
- **位置**：画面下部中央に統一
- **マージン**：画面下部から約20-30%の位置
- **幅**：画面幅の約80-90%
- **中央揃え**：すべてのテキストを中央揃え

#### 背景・可読性
- **背景色**：半透明の黒（RGB: 0, 0, 0）
- **透明度**：60-70%
- **背景形状**：矩形（角丸オプション）
- **パディング**：テキストの上下左右に適度な余白

#### フォント・テキスト
- **フォント**：サンセリフ体（例：Helvetica, Arial, Roboto）
- **ウェイト**：太め（Bold、700以上）
- **色**：白色（RGB: 255, 255, 255）
- **サイズ**：画面の約15-20%の高さ
- **行間**：1.2-1.5倍
- **文字間隔**：標準

#### アニメーション・効果
- **出現効果**：フェードイン（0.3-0.5秒）
- **タイミング**：音楽のビートに同期（推奨）
- **消滅効果**：フェードアウト（0.3-0.5秒）

## 🎵 音声・音楽・効果音の仕様

### BGM（背景音楽）

#### 音源特性
- **ジャンル**：オリジナル音源（制作者による独自制作）
- **テンポ**：中程度（推定BPM 90-120）
- **長さ**：動画総尺に合わせて18-24秒
- **ループ**：シームレスループ対応

#### 音楽の役割
- **感情表現**：ポジティブ、エネルギッシュ
- **テンポ感**：視聴者の注意を引き、離さない
- **ブランド化**：オリジナル音源で差別化を実現

### 効果音

#### タイミング
- **テキスト出現時**：軽いポップ音または効果音
- **シーン切り替え時**：トランジション効果音（オプション）
- **クロージング時**：終了を示す効果音（オプション）

#### 音質
- **音量**：BGMより小さく、邪魔にならない程度
- **種類**：軽く、ポジティブな印象の音
- **長さ**：0.1-0.3秒程度

### 音量バランス
- **BGM**：全体の70-80%
- **効果音**：全体の20-30%
- **ボーカル**：なし（テキストのみ）

## 🎯 制作時のポイント

### 重要な要素

1. **AIDA モデルの厳守**：感情的訴求 → 興味喚起 → 欲望醸成 → 行動喚起
2. **シーン1と2-5の対比**：感情的 vs. 理性的、屋外 vs. 室内、笑顔 vs. 落ち着き
3. **テキストの統一性**：すべてのテキストが同じ配置、背景、フォントを使用
4. **音楽のテンポ感**：視聴者の注意を引き、離さない
5. **視聴者への直接訴求**：カメラ目線で視聴者に直接訴求

### 応用のコツ
