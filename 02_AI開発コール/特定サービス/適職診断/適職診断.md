# 📋 夜のお店適職診断アプリ - 完全構築プロンプト
## 🎯 メインプロンプト

```
【AIデベロッパー向け】インスタ映えする夜のお店適職診断Webアプリを構築してください。

## 📌 プロジェクト概要
- **目的**: Instagram投稿用の夜のお店（ナイトワーク）適職診断アプリ
- **診断形式**: 10問の質問に答えて最適な店舗タイプを判定
- **結果タイプ**: 5種類（高級クラブ/ラウンジ/キャバクラ/ガールズバー/コンカフェ）
- **出力形式**: 診断結果を1080x1080pxのインスタ投稿用画像として生成・ダウンロード可能

### 2. 診断質問（10問）
以下の質問を実装してください：

**Q1: あなたの性格に最も近いのは？**
- 落ち着いていて上品な雰囲気
- 明るく華やかで人を惹きつける
- 親しみやすくフレンドリー
- 個性的でユニークな魅力がある

**Q2: お客様との接し方で理想的なのは？**
- 洗練された会話で心地よい時間を提供
- 楽しく盛り上げて笑顔にする
- 気軽に話せる友達のような関係
- テーマやキャラクターに沿った接客

**Q3: 自分の強みだと思うのは？**
- 容姿と雰囲気
- トークスキルと明るさ
- 親しみやすさと素直さ
- 個性と創造性

**Q4: 希望する年収は？**
- 1000万円以上
- 500万円〜1000万円
- 300万円〜500万円
- 200万円〜300万円

**Q5: どんなお客様と話したい？**
- 経営者や富裕層
- ビジネスマンや会社員
- 幅広い年齢層
- 若い世代やオタク層

**Q6: 働きたい年齢層は？**
- 30代後半以上
- 25歳〜35歳
- 20歳〜30歳
- 18歳〜25歳

**Q7: 夜の仕事の経験は？**
- 高級店での経験あり
- 一般的な夜の仕事の経験あり
- 未経験だけど興味あり
- 完全未経験

**Q8: 理想的なお店の雰囲気は？**
- 最高級の会員制クラブ
- 落ち着いた高級ラウンジ
- 華やかで活気のあるキャバクラ
- カジュアルで気軽なガールズバー
- テーマ性のあるコンカフェ

**Q9: どのくらいの頻度で働きたい？**
- 週1〜2回でしっかり稼ぐ
- 週3〜4回でバランスよく
- 週5回以上でガッツリ
- 自由に調整したい

**Q10: 最終的な目標は？**
- トップキャストとして名を馳せる
- 安定して高収入を得る
- 楽しく働きながら稼ぐ
- 自分らしく無理なく働く

### 3. スコアリングロジック

- `club`: 高級クラブ適性
- `lounge`: ラウンジ適性
- `cabaret`: キャバクラ適性
- `girlsbar`: ガールズバー適性
- `concept`: コンカフェ適性
