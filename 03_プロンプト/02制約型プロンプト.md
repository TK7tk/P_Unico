# 詳細説明
- 文字数制限
- 形式指定
- トーン指定
- 対象者限定などの制約条件を設けてAIの出力を制御する技法。品質の一貫性を保ち、特定の要件に適合した回答を得ることができます。ビジネス文書作成や標準化が必要な場面で特に有効な手法です。

# テンプレート

**制約条件設定**
- 文字数: `[具体的な文字数]以内`
- 形式: `[箇条書き/段落/表形式など]`
- トーン: `[丁寧/カジュアル/専門的など]`
- 対象者: `[専門家/初心者/一般向けなど]`

**タスク指示**
上記制約に従って`[具体的なタスク]`を実行してください。

**例**
```
300文字以内・箇条書き形式・丁寧な口調で、
新商品の特徴を3つ説明してください。
```

# 例示
環境問題の解決策提案

以下の制約条件に従って環境問題の解決策を提案してください：

## 制約条件
- 文字数: 200文字以内
- 提案数: 3つの解決策
- 形式: 番号付きリスト（1. 2. 3.）
- 追加情報: 各解決策の実現可能性を`一言で併記`

## 例
1. 再生可能エネルギーの推進 `→実現可能性: 高`
